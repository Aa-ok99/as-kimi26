# Installation Guide

## Termux Setup

1. Install Termux from Play Store or F-Droid
2. Update packages:
   ```
   pkg update && pkg upgrade
   ```
3. Install Node.js:
   ```
   pkg install nodejs
   ```
4. Clone repository:
   ```
   git clone https://github.com/agent-teams/as-kimi26.git
   cd as-kimi26
   ```
5. Install dependencies:
   ```
   npm install
   ```
6. Run setup:
   ```
   npm run setup
   ```
